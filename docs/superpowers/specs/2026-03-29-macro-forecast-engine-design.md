# 宏观预测引擎设计文档

**日期**: 2026-03-29
**项目**: Macro — 宏观预测引擎

---

## 1. 目标

基于宏观数据构建分层预测引擎，为11个资产标的提供方向性判断和量化评分，最终集成到周报中。

## 2. 预测标的

S&P 500、纳斯达克、道琼斯、上证指数、创业板、恒生指数、恒生科技、黄金、白银、美元指数、原油

## 3. 分层架构

### Layer 1: 宏观环境定位（增长×通胀四象限）

Z-score 标准化法。每个指标计算 vs 5年滚动均值的 Z-score，按类别加权平均。

**美国增长指标：** LEI、NFP 3月均值、初请（反向）、零售YoY、消费信心、工业生产YoY
**美国通胀指标：** 核心PCE YoY、粘性CPI、平均时薪YoY、截尾均值PCE
**美国政策指标：** 联邦基金利率3月变化、收益率曲线3m10y、M2 YoY

中国同样独立一套，加"美国传导"修正（±10%）。

四象限：Goldilocks / Reflation / Stagflation / Deflation

### Layer 2: 多因子评分卡

每个资产 6-10 个因子，打分 -2 到 +2，加权求和归一化到 -1~+1。

核心因子关系（基于 sell-side 验证）：
- 美股：ISM方向 + 信用利差 + 美联储政策 + 收益率曲线
- 黄金：10Y实际利率(反向) + 美元(反向) + 衰退概率
- 白银：黄金beta + ISM PMI(工业需求)
- A股：信贷脉冲(领先3-6月) + PMI + M1 + LPR政策
- 港股：中国信贷脉冲 + 美国利率(双beta)
- 美元：利差 + 美国增长相对优势
- 原油：全球增长 + ISM新订单

### Layer 3: 统计回测

简单回测：每月末打分 → 对比下月涨跌 → 统计胜率
严格回测（后续）：滚动窗口 + out-of-sample IC

### Layer 4: 报告输出

仪表盘总览 + 每个资产详细分析页

## 4. 时间维度

- 短期：下周展望（1周）
- 中期：未来1-3个月展望

## 5. 评分解读

| 综合评分 | 信号 |
|---------|------|
| > 0.6 | 强烈看多 |
| 0.2 ~ 0.6 | 偏多 |
| -0.2 ~ 0.2 | 中性 |
| -0.6 ~ -0.2 | 偏空 |
| < -0.6 | 强烈看空 |
