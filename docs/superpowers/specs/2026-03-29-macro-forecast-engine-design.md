# 宏观预测引擎设计文档 (Current State)

**日期**: 2026-03-29
**状态**: v1.0 已实现

---

## 1. 架构概述

分层预测引擎，从宏观数据到资产方向判断：

```
Layer 1: 宏观指标预测 (回归驱动)
    ↓
Layer 2: 宏观环境定位 (增长×通胀四象限)
    ↓
Layer 3: 资产多因子评分 (11个标的)
    ↓
Layer 4: 报告输出 (仪表盘 + 因子明细 + 共识对比)
```

## 2. 预测标的 (11个)

S&P 500, 纳斯达克, 道琼斯, 上证指数, 创业板, 恒生指数, 恒生科技, 黄金, 白银, 美元指数, 原油

## 3. Layer 1: 宏观指标预测

### 实现的回归模型

| 指标 | 方程 | 方法 | 质量 |
|------|------|------|------|
| CPI | 7分项加权 | 自下而上，实时汽油价格，±5%能源cap | ✅ 好 |
| 失业率 | UE = f(Claims, V/U ratio) | OLS回归，lag 1-2月 | ✅ R²=0.86 |
| 消费者信心 | Sent = f(Gas, UE) | OLS回归 | ✅ R²=0.60 |
| 联邦基金利率 | Taylor Rule | r*=0.5, NAIRU=4.0, Okun=2.0 | ✅ 规则驱动 |
| NFP | 3月加权均值 + 初请调整 | 趋势 (回归R²<0.15放弃) | ⚠️ 低信度 |
| 零售销售 | 3月加权趋势 | 趋势 (回归R²<0.15放弃) | ⚠️ 低信度 |
| 中国PMI | AR(1) + 信贷方向 | 趋势+规则 (回归R²<0.15放弃) | ⚠️ 中信度 |
| 中国CPI | PPI(t-2)回归 | OLS + 趋势回退 | ⚠️ 中信度 |

### 回归引擎 (_fit_and_predict)
- np.linalg.lstsq (无sklearn依赖)
- 自动日期对齐、滞后处理
- R² < 0.15 自动回退到加权趋势
- 输出: 系数、R²、t统计量、置信度

## 4. Layer 2: 宏观环境定位

### 增长×通胀四象限

| | 高通胀 | 低通胀 |
|---|---|---|
| **高增长** | Reflation (再通胀) | Goldilocks (金发女孩) |
| **低增长** | Stagflation (滞胀) | Deflation (通缩/衰退) |

### 评分方法
- Z-score 标准化 (vs 5年滚动均值)
- 增长: LEI, NFP, 初请(反向), 零售YoY, 消费信心, 工业生产YoY
- 通胀: 核心PCE YoY, 粘性CPI, 时薪YoY, 截尾均值PCE
- 政策: 联邦基金利率变化, 收益率曲线, M2 YoY
- 中国: PMI, 工业增加值, 零售, 信贷 + 美国传导修正(±10%)

### 前瞻象限
用 Layer 1 预测值替换当前值，重新跑象限定位 → 得到"未来象限"

## 5. Layer 3: 资产多因子评分卡

### 评分体系
- 每个因子: -2 到 +2
- 加权求和，归一化到 -1 ~ +1
- \>0.6 强烈看多, 0.2~0.6 偏多, -0.2~0.2 中性, -0.6~-0.2 偏空, <-0.6 强烈看空

### 核心因子关系

| 资产 | 第一驱动力 | 第二驱动力 | 第三驱动力 |
|------|-----------|-----------|-----------|
| 美股 | 增长象限+信用利差 | 美联储政策方向 | 收益率曲线 |
| 黄金 | 实际利率(反向) | 美元(反向) | 衰退概率 |
| 白银 | 黄金beta | ISM PMI(工业需求) | 实际利率 |
| A股 | 信贷脉冲(领先3-6月) | PMI + M1 | LPR政策 |
| 港股 | 中国信贷脉冲 + 美国利率(双beta) | 南向资金 | — |
| 美元 | 利差 | 美国增长相对优势 | — |
| 原油 | 全球增长 | ISM新订单 | 地缘 |

### 当前 + 前瞻双版本
- 当前评分: 基于最新实际数据
- 前瞻评分: 基于 Layer 1 预测值，反映"我们认为方向会怎么变"
