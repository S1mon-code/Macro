# Sell-Side 宏观研报方法论研究

> 基于 Goldman Sachs、JPMorgan、Morgan Stanley、BofA、Citi、Barclays、Deutsche Bank、UBS、CICC 等顶级投行的研究方法论总结。

## 一、报告结构（行业通用）

### 旗舰产品

| 投行 | 旗舰报告 | 特色 |
|------|---------|------|
| Goldman Sachs | US Economics Analyst（周报）、Macro Outlook（年度） | Current Activity Indicator (CAI)，Top of Mind 主题深度 |
| JPMorgan | Guide to the Markets（季度 ~75页） | 一页一图一洞察，视觉标杆 |
| Morgan Stanley | Bluepapers（长期主题）、Big Debates（年度） | 共识分歧点分析 |
| BofA | Global Fund Manager Survey、The Flow Show | Bull & Bear Indicator，50+自有指标 |
| Barclays | Global Economics Weekly | 60+经济学家全球覆盖 |
| UBS | 情景分析驱动 | 新旧经济分拆（新经济占GDP 15-20%） |
| CICC | 深度政策信号解读 | 央行/政府背景经济学家，TSF 分解 |

### 标准报告板块

1. **执行摘要 / 关键要点**（1页）

4. **通胀分析**（CPI/PCE 拆解、预测路径）
5. **就业市场评估**

7. **金融条件评估**
8. **风险情景**（牛/熊/基准 + 概率权重）
9. **数据日历 / 下周预览**
10. **预测修正追踪**（vs 上期变化）

### 报告长度参考

| 类型 | 页数 | 频率 |
|------|------|------|
| 日报 | 1-2页 | 数据反应 |
| 周报 | 3-8页 | 主题 + 数据回顾 |

## 二、美国宏观分析框架

### 2.1 经济周期判断

**NBER "3D" 框架：**
- **Depth**（深度）：衰退程度
- **Diffusion**（扩散）：波及范围
- **Duration**（持续时间）：持续多久
- 参考指标：实际个人收入（减转移支付）、非农就业、家庭调查就业、实际个人消费、制造业和贸易销售、工业生产

**领先/同步/滞后指标体系：**

| 类型 | 指标 |
|------|------|
| **领先** | 会议局 LEI（10个组成部分）、ISM 新订单-库存、建筑许可、收益率曲线、初请失业金、S&P 500、信用利差、消费者预期 |
| **同步** | 非农就业、工业生产、实际个人收入（减转移支付）、制造业和贸易销售 |
| **滞后** | 失业率（峰值滞后于衰退开始）、CPI（延迟响应）、平均失业时长、银行优惠利率 |

**衰退概率判断规则：**

| 规则 | 阈值 | 准确率 |
|------|------|--------|
| **Sahm Rule** | 失业率3月均值较12月低点上升 ≥0.50pp | 1970年以来每次衰退都命中 |
| **收益率曲线倒挂（3m10y）** | 倒挂持续 >3个月 | 73% |
| **ISM 新订单-库存** | 连续3个月为负 | 1970年以来每次衰退 |
| **会议局 LEI "3D" 规则** | 6月扩散指数 ≤50 且年化增速 < -4.3% | 领先拐点约7个月 |
| **GS 衰退概率模型** | 综合收益率曲线+信贷+初请+消费信心+PMI | 当前约30% |

**GDP Nowcasting 模型：**
- **Atlanta Fed GDPNow**：聚合13个GDP子项，月内更新6-7次
- **NY Fed Nowcast**：动态因子模型
- **GS Current Activity Indicator (CAI)**：24个高频实际活动指标，以GDP等价表示

### 2.2 通胀分析方法论

**CPI/PCE 拆解框架：**

| 维度 | 分类 |
|------|------|
| 粘性 vs 弹性 | Atlanta Fed 粘性价格CPI vs 弹性价格CPI |
| 商品 vs 服务 | 核心商品（贸易敏感）vs 核心服务（劳动力成本驱动） |
| 住房 vs 非住房 | CPI 住房权重~34%，PCE 住房权重~15%（CPI/PCE 差异主因） |
| 四桶 PCE 分解 | ①住房服务 ②核心服务除住房(supercore) ③核心商品 ④食品能源 |

**美联储最关注的指标（优先级排序）：**

1. **核心 PCE** — 官方目标指标（2%年度目标）
2. **Supercore PCE**（核心服务除住房）— 鲍威尔称"最重要的类别"，重度劳动力成本驱动
3. **Dallas Fed 截尾均值 PCE** — 去除价格变动上下尾，减少噪音
4. **Cleveland Fed 中位 CPI/PCE** — 取第50百分位，极度抗异常值
5. **NY Fed 多变量核心趋势 (MCT)**

**通胀预测输入变量：**
- 工资增长：~4% 对应 2% 通胀（假设~2%生产率增长）
- 住房/租金：新签租金领先 OER 约12-18个月
- 进口价格：美元强弱和全球商品供给
- 通胀预期：密歇根大学、NY Fed 调查、5y5y breakevens
- 劳动力市场紧张度：V/U 比率、辞职率
- 供应链压力：NY Fed GSCPI
- 大宗商品价格传导

### 2.3 就业市场分析

**核心监控指标：**

| 指标 | 作用 | 关注阈值 |
|------|------|---------|
| JOLTS 职位空缺 | 劳动力需求；V/U 比率衡量紧张度 | V/U < 1.0 = 不再紧张 |
| 辞职率 | 工人信心 | < 2.0% = 信心下降 |
| Beveridge 曲线 | 空缺率 vs 失业率；外移=结构性错配 | — |
| Sahm Rule | 3月均值失业率上升 | ≥ 0.50pp = 衰退 |
| U-6 广义失业率 | 包含边际附着+非自愿兼职 | — |
| 25-54岁劳动参与率 | 结构性劳动力供给 | — |
| 平均时薪 + ECI | 工资压力；ECI 控制成分效应 | ~3.5% 对应 2% 通胀 |
| 初请失业金 | 实时裁员信号（4周均值平滑） | 持续 >25-30万 = 恶化 |
| 家庭 vs 企业调查 | 分歧信号测量问题 | — |
| 平均周工时 | 裁员前先削减工时的领先指标 | — |
| 临时工就业 | 领先指标（先招/先裁） | — |

**信号矛盾解读：**
- U-3 低但 U-6 高 → 非自愿兼职隐含闲置
- NFP 强但参与率平 → 结构性因素（老龄化）
- 辞职下降但招聘稳定 → 市场正常化，非崩塌
- JOLTS 下降但失业稳定 → Beveridge 曲线回归疫情前

### 2.4 美联储政策分析

**预测工具：**
- **Taylor Rule**：i = r* + π + 0.5(π - π*) + 0.5(y - y*)，约70%准确率
- **点阵图**：季度发布（3/6/9/12月），关注中位点 vs 市场定价差异
- **Fed Funds Futures / OIS 曲线**：CME FedWatch 逐会议概率
- **FOMC 声明逐字对比**："some" vs "many" vs "most"；"balanced" vs "tilted"
- **前瞻指引分类**："Odyssean"（承诺未来行动）vs "Delphic"（预测未来条件）

### 2.5 金融条件

**Goldman Sachs FCI（五大组成部分）：**
1. 政策利率
2. 长期无风险债券收益率（10年期国债）
3. 企业信用利差
4. 股票估值（S&P 500 衍生）
5. 贸易加权汇率

权重由 FRB/US 宏观模型确定，反映各变量对一年期 GDP 增长的影响。

**Chicago Fed NFCI：**
- 105个金融活动指标
- 三个子指数：风险、信贷、杠杆
- 正值 = 紧于平均；负值 = 松于平均

**信用利差阈值：**
- HY 利差从低点扩大 >300bps = 修正信号
- HY 利差 >1,000bps = 衰退级压力
- 信用利差通常在衰退前 3-6 个月扩大

### 2.6 跨资产信号

| 信号 | 含义 |
|------|------|
| VIX > 30 | 恐慌升高；> 40 = 危机 |
| 防御性板块跑赢周期性 | 晚周期/衰退风险 |
| 盈利修正宽度转负 | 经济收缩 |
| 铜金比下降 | 风险偏好下降/衰退担忧 |
| USD/JPY 快速走强（日元升值） | 套息交易平仓，风险事件 |

## 三、中国宏观分析框架

### 3.1 增长评估

**GDP 拆解：**
- 净出口贡献（近期季度贡献高达~70%）
- 家庭消费贡献（Q3 2024 仅29%）
- 房地产拖累（~2个百分点/年）
- "新经济" vs "旧经济" 分拆

**替代增长追踪：**

| 方法 | 来源 | 描述 |
|------|------|------|
| 李克强指数 | The Economist 构建 | 电力消耗40% + 铁路货运20% + 银行贷款40% |
| China CAT | 旧金山联储 | 8个非GDP指标主成分分析 |
| 稀疏PLS | 纽约联储 | 月度指标替代增长估计 |
| 卫星夜光 | 国防卫星 | 验证经济活动总量 |
| 贸易伙伴进口数据 | 各国海关 | 独立验证（不受中国统计局影响） |

### 3.2 信贷脉冲（最重要的领先指标）

**公式：** 信贷脉冲 = (新增信贷(t) - 新增信贷(t-12)) / GDP

**领先性：**
- 领先全球工业产出 **9-12个月**
- 领先全球 PMI 6-9个月
- 领先新兴市场股票回报 6-12个月
- 领先工业金属 3-6个月

**阈值：**

| 脉冲水平（% GDP） | 信号 |
|-------------------|------|
| > +5% | 强刺激，全球再通胀 |
| +2% ~ +5% | 有意义的宽松 |
| 0% ~ +2% | 温和支持 |
| -2% ~ 0% | 轻度收紧 |
| < -2% | 显著收缩 |

### 3.3 社会融资 (TSF) 分解

| 组成部分 | 占比 |
|---------|------|
| 人民币贷款 | ~60% |
| 政府债券 | ~15% |
| 企业债券 | ~12% |
| 影子银行 | ~8%（2016年峰值32.9%） |
| 股权融资 | ~3% |
| 外币贷款 | ~2% |

### 3.4 房地产分析（中国宏观核心）

**追踪指标：**
- 百强开发商销售（CRIC 数据）
- 销售面积 / 金额
- 新开工 / 竣工面积
- 土地出让金
- 70城房价指数
- 库存 / 去化月数
- 开发商杠杆（三条红线）

**关键阈值：**
- 库存 >18个月 = 压力（三四线城市现状）
- 库存 <12个月 = 健康（一线城市）
- 银行 30% 贷款暴露于房地产
- 拨备覆盖率 205% 提供缓冲

**三条红线（2020年8月）：**
1. 资产负债率 < 70%
2. 净负债率 < 100%
3. 现金/短期借款 ≥ 1.0
- 触碰0条 = 允许15%年债务增长；1条=10%；2条=5%；3条=0%（债务冻结）

### 3.5 货币政策工具层级

| 工具 | 功能 | 解读 |
|------|------|------|
| **7天逆回购 (OMO)** | 主导短期政策利率 | 变动 = 政策方向信号 |
| **MLF (1年)** | 中期借贷便利；正在弱化 | 次要政策利率 |
| **LPR** | 所有银行贷款基准（20家银行报价） | 直接影响房贷和企业借贷成本 |
| **RRR** | 存款准备金率 | 降准 = 更激进的量化宽松 |
| **SLF** | 常备借贷便利 | 利率走廊上限 |
| **PSL** | 抵押补充贷款 | 定向工具（棚改、基建） |

**关键流动性监控指标：** DR007（7天银行间回购利率）

### 3.6 M1/M2 剪刀差

- **M1 > M2 增速**：传统看多 — 企业活期存款活跃投资
- **M1 < M2 增速**：看空 — 资金流入定期/储蓄存款
- M1-M2 差领先 CSI 300 约 6-12个月
- **注意**：2024年末央行扩大 M1 定义（加入居民活期和数字人民币），新旧口径不可比

### 3.7 高频指标

| 指标 | 频率 | 追踪目标 |
|------|------|---------|
| 六大电厂日耗煤 | 日度 | 工业活动和电力需求 |
| 螺纹钢期货 | 日度 | 建筑活动 |
| 集装箱运价指数 (CCFI/CBFI) | 周度 | 出口活动 |
| 交通拥堵指数（高德/百度） | 日度 | 城市经济活动 |
| 百度迁徙指数 | 日度 | 劳动力流动、返工 |
| 30城商品房成交 | 日度/周度 | 实时房产销售 |
| 货运量 | 月度 | 最可靠的实物活动指标 |

### 3.8 中国数据特殊性

- **累计 vs 当月**：统计局发布大多数数据为年初至今累计值
- **1-2月合并**：春节扰动，需合并分析
- **两会政策信号**：GDP目标设定刺激下限，财政赤字率信号财政立场，专项债额度指示基建管道
- **政策语言**："稳健" vs "适度宽松" vs "适当宽松" — 逐字解读

## 四、视觉呈现最佳实践

### 4.1 核心设计原则

1. **信息密度与清晰度并重** — 紧凑排版但不牺牲可读性
2. **一致的视觉语言** — 相同颜色、相同约定、每页如此
3. **始终提供历史上下文** — 平均值、范围、百分位
4. **层次分明** — 最重要数字最大最醒目
5. **一图一洞察** — 每张图回答一个问题
6. **标注关键** — 直接在图表上标注事件、拐点、阈值
7. **预测透明** — 始终展示预期 vs 实际

### 4.2 颜色体系（行业标准）

| 颜色 | 用途 |
|------|------|
| 深蓝/海军蓝 | 主数据系列 |
| 中蓝 | 次要数据系列 |
| 灰色 | 历史数据、上期、次要系列 |
| 红色 | 收缩、负面惊喜、恶化、卖出 |
| 绿色 | 扩张、正面惊喜、改善、买入 |
| 橙色/琥珀色 | 警示、过渡、接近阈值 |
| 金色/黄色 | 高亮（Goldman 强调色） |

### 4.3 常用图表类型

| 图表 | 用途 |
|------|------|
| 折线图 | GDP/通胀/收益率时间序列（叠加实际/预测/共识） |
| 柱状图 | GDP 分解、beat/miss 分析、跨国比较 |
| 堆叠面积图 | 信贷增长分解、通胀贡献 |
| 热力图 | 跨资产相关性、月度指标仪表盘、年度资产回报 |
| 散点图 | 估值框架、菲利普斯曲线、公允价值分析 |
| 瀑布图 | GDP 增长桥、盈利修正分解、财政脉冲 |
| 雷达图 | 多因子国别比较、周期定位 |
| 扇形图 | GDP 预测分布（置信区间） |
| 红绿灯仪表盘 | 15-25个指标状态总结（●绿/●黄/●红 + 方向箭头） |

### 4.4 第一页必备要素

1. **GDP 增长**：实际 | 预测 | 共识
2. **通胀**：Headline CPI/PCE YoY、Core YoY + 方向箭头
3. **就业**：NFP 月变化/3月均值、失业率（当前+预测）
4. **政策利率**：当前联邦基金利率、预期终端利率、下次变动日期
5. **金融条件**：自有 FCI 指数（单一数值 + 方向）
6. **关键风险**：3-5个要点，按概率和影响排序
7. **市场快照**：S&P 500、10Y 收益率、原油 — 当前水平 + 周/月变化

### 4.5 数据呈现格式

**共识 vs 实际 vs 前值：**

| 指标 | 前值 | 共识 | 实际 | 惊喜 |
|------|------|------|------|------|
| CPI YoY | 2.6% | 2.5% | 2.4% | 🟢 -0.1pp |

**历史上下文表：**

| 指标 | 当前 | 10年均值 | 25年均值 | 最低 | 最高 | 百分位 |
|------|------|---------|---------|------|------|--------|

**情景分析：**

| 熊市 (20%) | 基准 (60%) | 牛市 (20%) |
|------------|-----------|-----------|
| GDP: -0.5% | GDP: +2.0% | GDP: +3.5% |

## 五、我们 Macro 报告的升级方向

基于以上研究，建议按优先级实施以下升级：

### P0（最高优先级）

1. **经济周期仪表盘** — 红绿灯式指标状态总结（Sahm Rule、ISM、收益率曲线、LEI 等）
2. **通胀深度拆解** — 加入 supercore PCE、粘性/弹性 CPI 分类
3. **就业市场仪表盘** — JOLTS V/U 比率、辞职率、初请趋势、Sahm Rule 监控
4. **中国信贷脉冲** — 从 TSF 数据计算

### P1（高优先级）

5. **历史上下文** — 每个指标加入 10年均值、百分位排名
6. **美联储政策追踪** — Taylor Rule 计算、点阵图 vs 市场定价
7. **金融条件指数** — 构建简化版 FCI
8. **收益率曲线分析** — 3m10y、2s10s 利差 + 衰退概率

### P2（中等优先级）

9. **中国房地产追踪** — 房价、销售、库存月数
10. **跨资产信号** — 铜金比、VIX、信用利差
11. **报告视觉升级** — 按 JPMorgan Guide to the Markets 风格重做
12. **情景分析框架** — 牛/熊/基准 + 概率权重

### P3（长期）

13. **GDP Nowcasting** — 构建自己的活动指标
14. **数据惊喜指数** — 实际 vs 共识追踪
15. **中国高频数据** — 电厂煤耗、螺纹钢、集运运价
